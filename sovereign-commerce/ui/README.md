# UI Components for Sovereign Commerce Platform

This directory contains all frontend components for the Sovereign Commerce platform designed for diaspora funders.

## Structure

- `templates/` - HTML templates with ceremonial styling
- `static/` - CSS, JavaScript, and asset files
- `components/` - Reusable UI components
- `assets/` - Images, fonts, and other media

## Templates

- `index.html` - Landing page with sovereign welcome
- `catalog.html` - Sacred offerings catalog
- `dashboard.html` - Funder management portal
- `base.html` - Base template with ceremonial navigation

## Styling

The platform uses a mythic design system with:
- Sacred Gold (#D4AF37) for primary elements
- Sovereign Purple (#6B46C1) for secondary elements
- Ceremonial gradients and ethereal effects
- Bootstrap 5 foundation with custom overrides

## JavaScript

- `sovereign.js` - Core platform functionality
- `catalog.js` - Shopping cart and product management
- `dashboard.js` - Funder dashboard interactions

## Audience

Designed specifically for:
- Diaspora Funders
- Heirs and Councils
- Custodians
- Flamekeepers
